# Анализ упоминаний компаний в новостях и экспорт в Excel

## Описание

Добавлена новая функциональность для автоматического обнаружения упоминаний компаний в новостях и экспорта результатов в Excel файл с несколькими листами.

## Новые возможности

1. **Автоматическое обнаружение компаний** в заголовках и описаниях новостей
2. **Экспорт в Excel** с тремя листами:
   - "Компании" - все компании из базы данных
   - "Новости по компаниям" - новости, сгруппированные по компаниям
   - "Статистика" - сводная информация по анализу
3. **Командный интерфейс** для удобной работы через командную строку

## Файлы

- `company_analyzer.py` - логика анализа новостей на предмет упоминаний компаний
- `export_excel.py` - функции экспорта данных в Excel с красивым форматированием
- `cli.py` - удобный интерфейс командной строки
- `storage.py` - дополнена новыми функциями работы с данными
- `main.py` - обновлен с поддержкой аргументов командной строки

## Установка зависимостей

Для работы экспорта в Excel требуется установить библиотеку `openpyxl`:

```bash
pip install openpyxl
```

## Использование через командную строку

### 1. Парсинг новостей

```bash
# Базовый вариант (7 дней, 200 новостей на источник)
python cli.py parse

# С дополнительными параметрами
python cli.py parse --days 14 --limit 100 --export-csv

# Через основной скрипт (обратная совместимость)
python main.py parse --days 7 --limit 200
```

### 2. Анализ новостей на предмет упоминаний компаний

```bash
# Базовый анализ
python cli.py analyze

# Анализ с экспортом в Excel
python cli.py analyze --export-excel

# Анализ с ограничением и выводом топ-20 компаний
python cli.py analyze --limit 500 --top 20 --export-excel

# Через основной скрипт
python main.py analyze --export-excel --show-top 20
```

### 3. Экспорт данных

```bash
# Экспорт в Excel (анализ + компании)
python cli.py export --format excel --limit 1000

# Экспорт в CSV (только новости)
python cli.py export --format csv

# Экспорт в оба формата
python cli.py export --format both
```

### 4. Поиск новостей по компании

```bash
# Поиск новостей по конкретной компании
python cli.py search --company "Газпром"

# С ограничением результатов
python cli.py search --company "Сбербанк" --limit 20
```

### 5. Статистика

```bash
# Получение статистики по базе данных
python cli.py stats

# Через основной скрипт
python main.py stats --show-latest 10
```

### 6. Инициализация базы данных

```bash
# Инициализация или обновление структуры БД
python cli.py init
```

## Использование через Python

### Пример 1: Анализ новостей

```python
from company_analyzer import get_news_with_companies_from_db, get_all_companies_from_db
from export_excel import export_analysis_to_excel

# Получить анализ новостей
company_news = get_news_with_companies_from_db(limit=1000)

# Получить все компании из базы
all_companies = get_all_companies_from_db()

# Экспортировать в Excel
excel_path = export_analysis_to_excel(company_news, all_companies)
print(f"Файл создан: {excel_path}")
```

### Пример 2: Обнаружение компаний в тексте

```python
from company_analyzer import detect_companies_in_text

text = "Газпром и Сбербанк объявили о сотрудничестве с Apple"
companies = detect_companies_in_text(text)

for company, count in companies:
    print(f"{company}: {count} упоминаний")
```

## Формат Excel файла

Созданный Excel файл содержит три листа:

### 1. Лист "Компании"
- Название компании
- Источник данных
- Рейтинг
- Год
- Ссылка
- Дата добавления

### 2. Лист "Новости по компаниям"
- Компания
- Количество упоминаний
- Заголовок новости
- Источник новости
- Дата публикации
- Ссылка на новость
- Сниппет текста с упоминанием компании

### 3. Лист "Статистика"
- Компания
- Количество новостей
- Общее количество упоминаний
- Дата последней новости

## Тестирование

Для проверки работы создан тестовый скрипт:

```bash
python test_company_analysis.py
```

Этот скрипт:
1. Тестирует обнаружение компаний в тексте
2. Создает тестовые новости
3. Анализирует их на предмет упоминаний компаний
4. Экспортирует результаты в Excel

## Особенности работы

1. **Поиск с учетом морфологии**: Поиск ведется по нижнему регистру с учетом границ слов
2. **Игнорирование дубликатов**: При сохранении новостей игнорируются дубликаты по URL
3. **Автоматическое форматирование**: Excel файлы создаются с красивым форматированием и цветовым кодированием
4. **Логирование**: Все операции логируются с указанием времени и уровня важности

## Обработка ошибок

- При отсутствии модуля `openpyxl` будет выведено сообщение с инструкцией по установке
- Ошибки при парсинге отдельных источников не прерывают работу всей системы
- Все операции с базой данных защищены обработкой исключений
